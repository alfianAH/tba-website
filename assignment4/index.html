Regular Expression atau biasa disingkat regex atau regexp juga disebut sebagai rational expression. Biasanya pola seperti itu digunakan oleh algoritma pencarian string untuk operasi "find" atau "find and replace" pada string, atau untuk validasi input. Ini adalah teknik yang dikembangkan dalam ilmu komputer teoretis dan teori bahasa formal.

Contoh-Contoh Pengaplikasian Regex

Berikut adalah contoh-contoh aplikasi dari Regular Expression:

  1. Pencarian file di sistem operasi

    2. Sebagai contoh, ketika anda ingin mencari file dengan nama abc.txt.r13131 atau file.java.r1293. Maka, untuk melakukan pencariannya, kalian dapat menuliskan kode ini di command line

    dir * /s/b | findstr \.r[0-9]+$

    Pada akhir command, kalian dapat memperhatikan bahwa sistem akan mencari file dengan akhir ".r" dan sejumlah simbol numerik yang berubah-ubah

  2. Search Engine

    3. Tentu kita sudah mengetahui beberapa search engine. Salah satu nya yang terkenal adalah Google. Contohnya ketika anda ingin mencari sebuah kata yang tidak mengandung unsur kata spesifik.

    Contoh: Alfian -Hamdani.
    Maka Google akan memberikan hasil pencarian yang menampilkan kata Alfian dan tidak menampilkan kata Hamdani.

  3. Parsing

    4. Parser menerjemahkan serangkaian simbol ke dalam abstraksi tingkat yang lebih tinggi seperti bahasa yang diformalkan (sering kali menggunakan tata bahasa yang mendasari untuk "memahami simbol"). Anda memerlukan parser untuk menulis bahasa pemrograman, sistem sintaks, atau editor teks anda sendiri

    Berikut adalah contoh mendapatkan nama fungsi pada python dengan menggunakan regular expression

    import re
code = '''
def f1():
    return 1
def f2()
    return 2
'''
print(re.findall('def ([a-zA-Z0-9_]+)', code))
# ['f1', 'f2']

  4. Sistem Highlight Syntax

    5. Seperti yang dicontohkan sebelumnya tentang bagaimana mendapatkan nama fungsi, selanjutnya adalah bagaimana menciptakan sebuah teks editor bagi programmer sehingga lebih enak dipandang. Yaitu dengan memberikan warna-warna yang berbeda pada sintaks, seperti memberikan warna merah pada nama fungsi, warna abu-abu pada komentar dan lain sebagainya. Setiap teks editor pasti memiliki fitur tersebut, agar programmer dapat dengan mudah melihat sintaks
