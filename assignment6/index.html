Berdasarkan komposisi bentuk ruas kiri dan kanan produksinya, Noam Chomsky mengklasifikasikan 4 tipe grammar, sebagai berikut:

  1. Tipe 0 dikenal sebagai Unrestricted Grammar (UG)
  2. Tipe 1 dikenal sebagai Context Sensitive Grammar (CSG)
  3. Tipe 2 dikenal sebagai Context Free Grammar (CFG)
  4. Tipe 3 dikenal sebagai Regular Grammar (RG)
